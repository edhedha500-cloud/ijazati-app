# إجازتي - تطبيق إدارة الإجازات والخصميات

تطبيق ويب متكامل لإدارة الإجازات والخصميات للموظفين، مصمم خصيصاً للدول العربية.

## الميزات الرئيسية

### 📱 واجهة مستخدم عربية
- تصميم متجاوب يعمل على الهواتف المحمولة (عرض أقصى 480 بكسل)
- دعم كامل للغة العربية (RTL)
- خط Tajawal الأنيق
- نظام لوني جذاب بتدرجات بنفسجية وبرتقالية

### 🏠 الصفحة الرئيسية
- تقويم تفاعلي يعرض أيام الشهر مع ألوان دلالية
- بطاقات إحصائية تعرض:
  - رصيد الإجازات المستحق
  - رصيد الإجازات المتبقي
  - خصم الشهر
  - رصيد الإجازات المستهلك
  - عدد مرات التأخير والانصراف
- نظام تصفية وبحوث
- زر إضافة عائم (FAB) للوصول السريع

### 👤 بيانات المستخدم
- تسجيل الاسم الكامل ورقم الهاتف والدولة
- بطاقة ترحيبية جذابة

### ⚙️ الإعدادات
- الراتب الشهري والعملة
- ساعات الدوام اليومي
- نوع الدوام (فترة واحدة / فترتين)
- أوقات البدء والانتهاء
- دورة الشهر (بداية ونهاية)
- أيام الإجازة الأسبوعية
- سياسة ترحيل الرصيد

### 🏖️ إدارة الإجازات
- تسجيل إجازات جديدة بأنواع متعددة
- التحقق من التداخل مع الإجازات الأخرى
- التحقق من أيام الإجازات الأسبوعية
- أنواع الإجازات:
  - سنوية، بدون راتب، راحة بالخصم (تخصم من الرصيد)
  - طارئة، مرضية، مهمة عمل، عطلة رسمية (لا تخصم)
  - إمكانية إضافة أنواع مخصصة

### 🚶 إذن انصراف و ⏰ إذن تأخير
- تسجيل الانصراف والتأخير مع الوقت
- حساب الساعات تلقائياً
- التحقق من عدم وجود إجازة في نفس اليوم

### 💳 رصيد الإجازات
- الرصيد المرحل من السنة الماضية
- سياسات ترحيل متعددة
- جدول الرصيد الشهري قابل للتعديل
- حساب آلي للرصيد المرحّل

### 📊 التقارير
- نطاقات زمنية متعددة (أسبوع، شهر، سنة، مخصص)
- إحصائيات شاملة
- تصدير Excel (صفحات متعددة)
- تصدير PDF (طباعة)

### 🧮 حاسبة الخصميات
- ثلاثة أنماط إدخال: دقائق، ساعات، أيام
- عرض تفصيلي للحسابات
- حساب المبلغ المخصوم بناءً على الراتب

### 📝 سجل النشاط
- عرض جميع العمليات مع التاريخ والوقت
- ترتيب تنازلي (الأحدث أولاً)
- حد أقصى 300 سجل

## التقنيات المستخدمة

- **React 18** - مكتبة الواجهة
- **Vite** - أداة البناء والتطوير
- **localStorage** - التخزين المحلي
- **XLSX** - تصدير Excel

## التثبيت والتشغيل

```bash
# تثبيت التبعيات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء النسخة الإنتاجية
npm run build

# معاينة النسخة الإنتاجية
npm run preview
```
