# تطبيق إجازتي - دليل التطوير

## نظرة عامة
تطبيق ويب متكامل لإدارة الإجازات والخصميات للموظفين، مبني باستخدام React + Vite.

## الميزات المكتملة

### ✅ الصفحات
- **الصفحة الرئيسية**: لوحة تحكم مع بطاقات إحصائية وزر إضافة عائم
- **بياناتي**: نموذج إدخال بيانات المستخدم مع التحقق
- **الإعدادات**: إعدادات الراتب والدوام ودورة الشهر
- **رصيد الإجازات**: جدول شهري لحساب الرصيد والترحيل

### ✅ المكونات
- نظام مكونات كامل (Common.jsx)
- مكونات الصفحة الرئيسية (HomeComponents.jsx)
- التخطيط (Layout.jsx)

### ✅ المنطق البرمجي
- دوال مساعدة شاملة (helpers.js)
- إدارة التخزين المحلي
- حسابات الإجازات والخصومات
- التحقق من التداخل
- سياسات الترحيل

### ✅ التصدير
- تصدير Excel (متعدد الصفحات)
- تصدير PDF (باستخدام jsPDF)

## البنية التقنية

```
ijazati-app/
├── src/
│   ├── components/
│   │   ├── Common.jsx       # مكونات الواجهة العامة
│   │   ├── HomeComponents.jsx # مكونات الصفحة الرئيسية
│   │   └── Layout.jsx        # الشريط العلوي والقائمة الجانبية
│   ├── pages/
│   │   ├── UserDataPage.jsx  # صفحة بيانات المستخدم
│   │   ├── SettingsPage.jsx  # صفحة الإعدادات
│   │   └── BalancePage.jsx   # صفحة رصيد الإجازات
│   ├── utils/
│   │   └── helpers.js        # دوال مساعدة
│   ├── hooks/
│   │   └── useLocalStorage.js
│   ├── styles/
│   │   └── global.css
│   ├── App.jsx
│   └── main.jsx
├── package.json
├── vite.config.js
└── README.md
```

## التشغيل

```bash
# تثبيت التبعيات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء النسخة الإنتاجية
npm run build

# معاينة النسخة الإنتاجية
npm run preview
```
